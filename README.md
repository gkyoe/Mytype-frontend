# My Type 은 사용자가 원하는 유튜브 영상을 자신만의 카테고리로 관리 할 수 있는 웹 어플리케이션입니다.

## | 주요기능 |

- 로그인, 로그아웃, 회원가입
- 구글 소셜로그인
- 유튜브 영상 검색
- 추천 카테고리 영상 랜더
- 나만의 카테고리 생성
- 나만의 카테고리에 원하는 영상 추가 및 삭제

## | 주요 스택 |

- react, redux
